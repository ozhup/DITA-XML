Erste Einstellungen

Auf dieser Seite finden Sie Informationen zur erstmaligen Einrichtung des Scanners.

Um den Scanner erfolgreich einzurichten, führen Sie die folgenden Grundeinstellungen durch:
  1. Halten Sie die Einschalttaste 5 Sekunden lang gedrückt, um den Scanner einzuschalten. Sobald das LED-Licht blinkt, können Sie mit der Einrichtung beginnen.
  2. Scannen Sie einen Test-Barcode, um die Einrichtung zu überprüfen. Sie können einen Test-Barcode scannen, indem Sie den USB-Empfänger in einem Abstand von etwa 5 cm zum Code bewegen. Ein akustisches Signal bestätigt die korrekte Funktionsweise. Falls Sie kein Signal hören, wechseln Sie zum Abschnitt Problemlösung.
  3. Scannen Sie den bevorzugten Barcode, um die Sprache auszuwählen.
  4. Scannen Sie diesen Barcode, wenn Sie im drahtlosen Modus arbeiten möchten.

  5. Scannen Sie diesen Barcode, wenn Sie im kabelgebundenen Modus arbeiten möchten.
Der Scanner ist betriebsbereit und arbeitet im bevorzugten Modus. Folgen Sie den weiteren Anweisungen zur Nutzung.
